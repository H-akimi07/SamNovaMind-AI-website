# 🤖 SamNovaMind AI

**SamNovaMind AI** is a modern, responsive, and visually stunning AI technology company website built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**.

This project showcases a premium landing page with modern UI/UX, smooth animations, responsive layouts, and interactive components inspired by professional technology websites.
